Sitios Responsivos con Media Queries

El Diseño Web Responsivo o “Responsive Web Design” en ingles, es un término que se refiere a la capacidad de que un sitio o diseño web se adapte al tamaño de cualquier dispositivo.
media queries (en español "consultas de medios") son útiles cuando deseas modificar tu página web o aplicación en función del tipo de dispositivo.

El contenido se adapta al tamaño de la pantalla.
Los elementos se ajustan automáticamente para un desplazamiento vertical.
El menú del sitio también se adapta al dispositivo usado.
Es posible seleccionar los contenidos destacados para el móvil.
